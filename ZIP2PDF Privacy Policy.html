개인정보처리방침

[ZIP2PDF]는 사용자의 개인정보 보호를 최우선으로 고려합니다. 본 개인정보처리방침은 앱이 처리하는 데이터와 관련된 정보를 설명합니다.

1. 데이터 수집

본 앱은 어떠한 개인정보도 수집하지 않습니다. 사용자의 이름, 이메일 주소, 위치 정보, 기기 식별자 등 개인 식별 정보를 포함한 어떠한 데이터도 수집하거나 저장하지 않습니다.

2. 파일 접근

본 앱은 다음과 같은 데이터에 접근합니다:

  • ZIP 파일 및 이미지 파일: 사용자가 선택한 기기 내부 저장소의 ZIP 파일과 그 안의 이미지 파일(jpg, png, jpeg 등)에 접근하여 PDF로 변환합니다.
  • 이 접근은 오직 사용자가 명시적으로 선택한 파일에 한정되며, 앱은 다른 파일이나 폴더에 접근하지 않습니다.

3. 데이터 사용 목적

파일 접근은 다음과 같은 목적으로만 사용됩니다:

  • 사용자가 선택한 ZIP 파일 내 이미지 파일을 추출하여 PDF 문서로 변환.
  • 변환된 PDF 파일을 사용자의 기기 내부 저장소에 저장.

4. 데이터 저장 및 보호

  • 저장 위치: ZIP 파일, 이미지 파일, 생성된 PDF 파일은 사용자의 기기 내부 저장소에만 저장됩니다. 본 앱은 클라우드 서버나 외부 저장소에 데이터를 업로드하거나 전송하지 않습니다.
  • 보호 조치: 앱은 안드로이드의 권한 관리 시스템을 준수하며, 사용자가 명시적으로 허용한 파일에만 접근합니다.

5. 제3자와의 데이터 공유

본 앱은 사용자의 파일 데이터나 기기 정보를 제3자와 공유하지 않습니다. 사용자의 데이터는 기기 내부에서만 처리됩니다.

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 파일 접근 권한 철회: 안드로이드 설정에서 저장소 접근 권한을 비활성화하여 앱의 파일 접근을 차단할 수 있습니다.
  • 데이터 삭제: 앱이 생성한 PDF 파일은 사용자가 직접 삭제할 수 있습니다. 앱은 기기 내 데이터를 영구적으로 저장하지 않습니다.

7. 안드로이드 권한

본 앱은 다음과 같은 권한을 요청합니다:

  • 저장소 읽기/쓰기 권한 (READ_EXTERNAL_STORAGE, WRITE_EXTERNAL_STORAGE 또는 MANAGE_EXTERNAL_STORAGE): 사용자가 선택한 ZIP 파일과 이미지 파일에 접근하고 PDF 파일을 저장하기 위해 필요합니다.

권한 요청 시 사용 목적을 명확히 설명하며, 사용자의 동의 없이는 파일에 접근하지 않습니다.

8. 개인정보처리방침 변경

본 개인정보처리방침은 법적 요구사항이나 앱 기능 변경에 따라 업데이트될 수 있습니다. 변경 사항은 GitHub 리포지토리에서 공지됩니다.
